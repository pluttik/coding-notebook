Variable types

Variables are immutable, objects are mutable

var a = 2; //number

var b = 'word'; //string

var c = ['word', 'letter']; //array - in fact type object

var d = {first: 'a', last: 'z'}; //object

var e; //undefined

typeof null; //object

typeof function myFunction() {}; //function


More about objects

Objects have properties

Each property consists of name:value

Values can be functions - name:method

Access a property like this: objectName.propertyName

Access a method like this: objectName.methodName()

Three ways to create a new object:

  1. (single object) object literal: var a = {x:1, y:2}
  2. (single object) with the keyword new: var a = new Object(); a.x=1; a.y=2;
  3. define an object constructor:
    function A(x, y) { this.xhere = x; this.yhere = y; }
    var s = new A(1, 2);

Accessing names and values in an object:

var ab = {name1: 'value1', name2: 'value2'};

for (z in ab) { console.log(ab[z]); } //"value1" "value2"

Object.keys(ab) //"name1" "name2"

Object.values(ab) //"value1" "value2"

delete ab.name2;

Adding properties to an object:

  1. object.newName = newValue;
  2. object.newName = function () { return this.oldName + '1'; }

Getters and setters

Getters and setters are functions that work like properties

get and set are keywords

var person = { name: 'John',

get id() { return this.name; },

set newName(newName) { this.name = newName; }

};

Accessing getters and setters:

getter: person.id

setter: person.newName = 'Mary';


Constructors

A constructor is a function that creates an object

General form: function Person(x, y) {

this.xhere = x;

this.yhere = function() {};

}

Adding properties to a constructor:

*** not possible ***

Instead, use prototype: Person.prototype.zhere = z;


Classes

A class is a kind of constructor function

A class can have only one constructor method

A constructor can use the super keyword to call the constructor of the super class

General form: class Person {

constructor(x) {

this.xhere = x; } //name:value pairs go in the contructor

yhere() {

console.log(this.xhere); } //methods go here, into the prototype

}

Extends and super for inheritance to a subclass

General form: class Kid extends Person {

constructor(x) {

super(x); }

zhere() {

console.log(this.xhere + 'z'); } }

this

  1. this is the object that owns the code
  2. in an object, this is the object itself
  3. in a function, this is the owner of the function
  4. in a constructor function, this does not have a value yet (when a new object is created, this will become the new object)
