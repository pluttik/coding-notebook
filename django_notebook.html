General stuff

Getting started (Python 3 and Django 2)

python get-pip.py # install pip

pip install virtualenv # install virtualenv

path\python virtualenv myvenv # make a virtual environment

myvenv\scripts\activate # start up the virtual environment

pip install django # install django

django-admin startproject mysite # start a new django project

python manage.py runserver # check in browser that site is working


Create an app

python manage.py startapp myapp # start a new app

# add myapp to app list in settings.py

path('myapp/', include(('myapp.urls', 'myapp'), namespace='myapp')), # include link to app url file in main urls.py

# make urls.py file inside myapp's folder

path('myapp/', MyAppView.as_view(), name='url_name'), # include link to a view

class MyView(TemplateView): # make a view in views.py

template name = "my_template_name.html"

class MyModel(models.Model): # in models.py, make a model for some data to show

python manage.py makemigrations # prepare database design changes for migration

python manage.py migrate # migrate the (new parts of the) database model


Python shell

python manage.py shell # start the shell
